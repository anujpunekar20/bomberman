### instructions
* download the zip file "Bomberman"
* extract the files into a folder
* run the exe file name "bomberman"

### controls - player1
* W to move up
* A to move left 
* S to move down
* D to move right
* space to drop bomb

### controls - player2
* Up arrow to move up
* Left arrow to move left 
* Down arrow to move down
* Right arrow to move right
* return to drop bomb
